# FRED ADVISORY — Visa Consultancy Website

## Files
- index.html  → main website (single file, all code + main hero image embedded)
- img/        → destination photos for all country detail pages
- vercel.json → deployment config

## Deploy to Vercel
1. Upload ALL files keeping folder structure (index.html + img/ folder together) to GitHub
2. Import repo on Vercel → Framework: Other → Deploy
3. Add domain FREDADVISORY.com in Vercel Settings → Domains

## Features
- Tourlane-inspired hero, sections, and country detail pages
- 12 destination cards (click → full visa package page with photo gallery)
- Real destination photos for 10 countries
- Fully mobile responsive with hamburger menu
- WhatsApp consultation CTAs throughout
